# NAT-Tracks
NAT Track API

Returns a JSON response of all current North Atlantic Tracks in the format of:

1. NAT Identifier
2. TMI number
3. Route (includes parseable decimal coordinates)
4. Available flight levels
5. Direction (Enum value: 0 = Unknown, 1 = West, 2 = East)
